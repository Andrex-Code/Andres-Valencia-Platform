# Strata Civil

## Categoria
engineering

## Concepto
Plantilla premium para ingenieria civil, consultoria tecnica e infraestructura.
La direccion visual usa azul acero, cobre tecnico y patrones ligeros inspirados en
planos y topografia para sentirse mas firma tecnica que landing generica.
