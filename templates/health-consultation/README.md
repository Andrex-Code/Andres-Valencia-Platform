# Health Consultation

Plantilla base para psicologia, salud, bienestar y consulta profesional.

## Editor

- Abrir demo: `templates/health-consultation/index.html`
- Abrir editor: `templates/health-consultation/index.html?edit=1`
- Storage local: `av-template-editor::health-consultation`

## Secciones clave

- Hero principal
- Banda de confianza
- Servicios
- Sobre la profesional
- Proceso
- Preguntas frecuentes
- Cierre comercial
- Footer

## Notas
- Parte de la estructura de `monica-valencia`, pero queda como base generica para salud y consulta.
- La siguiente iteracion deberia terminar de despersonalizar servicios, FAQ y bloques de autoridad.
