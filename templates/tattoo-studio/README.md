# Sombra Ink - Tattoo Studio Template

## Resumen
Plantilla para estudio de tatuajes enfocada en una estética premium oscura, con CTA fuerte a WhatsApp.

## Estructura
- `index.html`: plantilla principal.

## Personalización rápida
1. Cambia textos y datos de contacto.
2. Reemplaza imágenes por las del negocio real.
3. Ajusta colores en variables CSS dentro de `:root`.
