# Reference Notes

Fuentes consultadas el 13 de abril de 2026 para elevar la direccion del portafolio y del editor:

## Direccion visual del portafolio
- Instrument Work: https://www.instrument.com/work/
  - Referencia para una portada work-first, posicionamiento corto y una presentacion donde los casos son la prueba principal.
- PORTO ROCHA / live: https://portorocha.com/ie/
  - Referencia para una direccion tipografica mas directa y para reducir decoracion superflua.
- BASIC/DEPT case study / American Express: https://www.basicagency.com/case-studies/american-express-design-sytem
  - Referencia para ordenar mejor la relacion entre sistema visual, claridad y valor de negocio.
- Build in Amsterdam / About: https://www.buildinamsterdam.com/about
  - Referencia para combinar un tono boutique con lenguaje estrategico claro.
- Build in Amsterdam / Powerhouse Company: https://www.buildinamsterdam.com/case/powerhouse-company
  - Referencia para pensar layouts editoriales y paginas de caso con narrativa limpia.

## Direccion del editor
- Webflow content editor: https://help.webflow.com/hc/en-us/articles/33961251014931-Edit-site-content-as-a-content-editor
  - Referencia para una interfaz corta y segura donde se edita contenido sin tocar el layout.
- Framer editing content tutorial: https://www.framer.com/marketplace/tutorials/editing-content/
  - Referencia para priorizar tareas reales del cliente: texto, imagenes, links y contenido.

## Publicacion
- Vercel docs: https://vercel.com/docs
  - Base para mantener el portafolio publico como sitio estatico y separar el studio de cualquier despliegue prematuro sin persistencia seria.
- Vercel Storage overview: https://vercel.com/docs/storage
  - Base para la decision de no desplegar el studio con writes locales hasta mover contenido e imagenes a storage duradero.

## Decision tomada para la home
- Evitar glassmorphism, slogans de agencia generica y exceso de bloques decorativos.
- Priorizar una home oscura, tipografica y centrada en casos, mas cerca de un portfolio editorial que de una landing aspiracional generada.
- Mantener el editor como prueba de producto, pero sin dejar que la portada se convierta en una demo tecnica fria.
