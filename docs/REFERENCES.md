# Reference Notes

Fuentes consultadas el 13 de abril de 2026 para elevar la direccion del portafolio y del editor:

## Direccion visual del portafolio
- Instrument: https://www.instrument.com/
  - Referencia para una portada work-first, posicionamiento corto y un tono de estudio digital serio.
- PORTO ROCHA: https://portorocha.com/about
  - Referencia para una presentacion de estudio directa, con menos ruido y mas claridad de criterio.
- BASIC/DEPT case study: https://www.basicagency.com/case-studies/heath-ceramics-ecommerce-web-design
  - Referencia para estructurar las tarjetas como mini casos con foco en deliverables y problema resuelto.
- Awwwards portfolio nominees: https://www.awwwards.com/websites/web-design-portfolio/
  - Referencia para validar una direccion mas editorial y menos generica en portfolios creativos.

## Direccion del editor
- Webflow content editor: https://help.webflow.com/hc/en-us/articles/33961251014931-Edit-site-content-as-a-content-editor
  - Referencia para una interfaz corta y segura donde se edita contenido sin tocar el layout.
- Framer editing content tutorial: https://www.framer.com/marketplace/tutorials/editing-content/
  - Referencia para priorizar tareas reales del cliente: texto, imagenes, links y contenido.

## Publicacion
- Vercel docs: https://vercel.com/docs
  - Base para mantener el portafolio publico como sitio estatico y separar el studio de cualquier despliegue prematuro sin persistencia seria.
- Vercel Storage overview: https://vercel.com/docs/storage
  - Base para la decision de no desplegar el studio con writes locales hasta mover contenido e imagenes a storage duradero.
