Metodo / 04

El proceso se cuenta como capas de lectura, no como pasos decorativos.

Cada franja representa una etapa de lectura para que la metodologia se vea propia del oficio y no como un paso decorativo.

Capa 01 / contexto

Se entiende el tipo de proyecto, la pregunta tecnica y el momento exacto en que aparece.

No es lo mismo revisar un lote que intervenir una excavacion activa.

Capa 02 / lectura

Se organizan antecedentes, observaciones de sitio y cualquier evidencia disponible sobre el terreno.

La calidad de la decision mejora cuando campo y necesidad del proyecto se leen juntos.

Capa 03 / criterio

Los datos se traducen en recomendaciones comprensibles para diseno, obra o revision documental.

El objetivo es bajar incertidumbre y ordenar el siguiente paso.

Capa 04 / acompanamiento

Si el proyecto lo requiere, se acompana la decision con soporte puntual durante ajustes o ejecucion.

La recomendacion puede conversar con obra, disenador o cliente segun el caso.
