FAQ

Preguntas que este portafolio deberia responder sin rodeos.

Que hace distinta a cada demo?

La direccion visual, el tono, la estructura narrativa y la logica comercial. No es una misma plantilla con colores nuevos.

El editor ya se puede usar?

Si. La demo de panaderia ya permite editar, guardar localmente, exportar e importar contenido sin depender de un backend para el portafolio publico.

Que queda como siguiente mejora grande?

Llevar el studio interno a almacenamiento persistente real para publicar una version multi-negocio sin depender del sistema de archivos local.
