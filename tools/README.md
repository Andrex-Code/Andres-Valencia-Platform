# Tools Locales

Esta carpeta es para utilidades portables que no quieres instalar como administrador.

## Python portable
Si en alguno de tus equipos no tienes Python instalado, copia aqui una version portable con esta ruta:

```text
tools/python/python.exe
```

Los scripts del proyecto la detectan automaticamente antes de intentar usar `python` o `py`.

## Nota
El contenido de `tools/python/` esta ignorado por Git para que no subas binarios pesados por accidente.
